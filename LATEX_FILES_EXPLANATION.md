# LaTeX文件说明

## 文件分类

### ✅ **必须保留的文件**

1. **`main.tex`** - LaTeX主源文件
   - 这是你的报告源代码
   - **绝对不能删除**，这是所有内容的来源

2. **`main.pdf`** - 最终生成的PDF报告
   - 这是最终提交的文件
   - **必须保留**，这是你的成果

### 📦 **可以删除的辅助文件**（编译时自动重新生成）

这些文件是LaTeX编译过程中自动生成的辅助文件，用于存储临时信息：

1. **`main.aux`** - 辅助文件
   - 存储交叉引用信息（如 `\ref{}`, `\label{}`）
   - 删除后下次编译会重新生成

2. **`main.log`** - 编译日志
   - 包含编译过程的详细信息
   - 用于调试编译错误
   - 删除后不影响PDF生成

3. **`main.lof`** - 图表目录文件
   - List of Figures
   - 存储图表目录信息
   - 删除后下次编译会重新生成

4. **`main.lot`** - 表格目录文件
   - List of Tables
   - 存储表格目录信息
   - 删除后下次编译会重新生成

5. **`main.out`** - 书签输出文件
   - 存储PDF书签信息
   - 删除后下次编译会重新生成

6. **`main.toc`** - 目录文件
   - Table of Contents
   - 存储目录信息
   - 删除后下次编译会重新生成

## 建议

### 可以安全删除的文件

如果你已经生成了最终的PDF，可以删除以下文件以清理项目：

```bash
# 删除所有LaTeX辅助文件
rm main.aux main.log main.lof main.lot main.out main.toc
```

**注意**：
- 删除这些文件后，下次编译LaTeX时会自动重新生成
- 第一次编译可能需要运行两次才能正确生成目录和交叉引用
- 如果遇到交叉引用问题，删除这些文件后重新编译通常可以解决

### 保留的文件

- ✅ `main.tex` - 源代码
- ✅ `main.pdf` - 最终PDF报告
- ✅ `sections/` - 所有章节源文件
- ✅ `visualizations/` - 所有图表文件

## 清理脚本

如果需要，可以创建一个清理脚本来删除这些辅助文件。
