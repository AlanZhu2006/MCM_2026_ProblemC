# 模型准确率分析与改进指南

## 当前准确率：51.84%

### 准确率含义

**准确率 = 正确预测的周数 / 总周数**

- **总周数**: 299周
- **正确预测**: 155周
- **错误预测**: 144周
- **准确率**: 51.84%

这意味着：使用估计的粉丝投票计算综合得分后，能够正确预测约52%的周次的淘汰结果。

## 为什么准确率不是100%？

### 1. **逆问题的多解性**
- 从已知淘汰结果反推粉丝投票是一个逆问题
- 可能存在多个不同的粉丝投票组合都能产生相同的淘汰结果
- 模型选择了其中一个解，但不一定是"真实"的解

### 2. **模型简化假设**
- 当前模型假设粉丝投票与评委评分有一定相关性
- 但现实中，粉丝投票可能受很多因素影响（人气、争议性、个人魅力等）
- 这些因素难以量化

### 3. **数据限制**
- 某些周次可能无法准确确定被淘汰选手
- 某些周次可能有特殊规则（如评委选择、双淘汰等）
- 数据中可能包含异常值

### 4. **优化算法限制**
- 使用局部优化算法（SLSQP），可能陷入局部最优
- 约束条件可能过于严格或不够准确
